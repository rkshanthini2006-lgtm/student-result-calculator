# Student Result Calculator

A simple Student Result Calculator built using HTML, CSS, and JavaScript.

## Features

- Student Name
- Register Number
- Five Subject Marks
- Total Marks
- Average
- Percentage
- Grade Calculation
- Pass / Fail Result
- Input Validation
- Reset Button
- Print Result
- Generated Date & Time
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
